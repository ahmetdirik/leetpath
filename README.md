# LeetPath

AI-powered NeetCode 150 progress tracker with pattern-based weakness detection and personalized daily problem recommendations.

> 🚧 In active development. Coming soon.

## What it does

Most coding interview prep tools work at the single-problem level — they help you solve one problem at a time. LeetPath operates at the meta-layer: it analyzes patterns across your submissions to detect where you're consistently weak, then recommends the next problem to work on with reasoning.

## Tech stack

- **Backend:** FastAPI + SQLAlchemy 2.0 + PostgreSQL
- **Frontend:** Jinja2 + HTMX + Alpine.js + Tailwind CSS
- **Infrastructure:** Docker + Fly.io + Neon + Upstash
- **AI layer:** DeepSeek V4 Pro via OpenCode Pro
